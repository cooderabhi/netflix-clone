# Netflix Clone Backend 🎬

A Node.js + Express backend for a Netflix-style application, featuring secure user authentication and scalable architecture.

## 🚀 Features

* User Registration & Login
* Password hashing using bcrypt
* JWT-based authentication
* Clean and modular structure
* Error handling and validation

## 🛠 Tech Stack

* Node.js
* Express.js
* JWT (Authentication)
* bcrypt (Security)

## 📁 Folder Structure

```
server/
 ├── controllers/
 ├── routes/
 ├── middleware/
 ├── models/
 └── server.js
```

## ⚙️ Setup Instructions

1. Clone the repo

```
git clone https://github.com/cooderabhi/netflix-clone-backend.git
```

2. Install dependencies

```
npm install
```

3. Run the server

```
npm run dev
```

## 🔐 Environment Variables

Create a `.env` file:

```
JWT_SECRET=your_secret_key
```
